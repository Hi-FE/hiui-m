# Btn

> 按钮组件

## API

| 参数           | 说明                      | 类型        | 默认值     |
| ------------ | :---------------------- | --------- | ------- |
| size         | 按钮尺寸，可选值 `sm`/`md`/`lg` | `String`  | `md`    |
| disabled     | 是否禁用                    | `Boolean` | `false` |
| is_loading   | 是否加载中                   | `Boolean` | `false` |
| loading_text | 加载时的文本                  | `String`  | -       |
| prefix       | 前置图标                    | `String`  | -       |
| suffix       | 后置图标                    | `String`  | -       |
