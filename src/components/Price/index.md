# Price

> 价格

## API

| 参数 | 说明 | 类型 | 默认值 |
| ----|:-----| ---- | ---- |
| color | 颜色 | `String` | `#f02f45` |
| font_size | 字号大小 | `Number` | `24` |
| super | 价格符号是否对其文本的上标  | `Boolean` | `false` |
| delete | 是否为`Del`样式  | `Boolean` | `false` |
| suffix | 后缀文本  | `String/Number` | - |
| text_size | 文本和符号大小  | `Number` | - |
| no_shrink | 文本和符号价格符号不缩小  | `Boolean` | `false` |
| symbol | 价格符号  | `String` | `￥` |
